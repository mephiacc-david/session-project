## Мониторинг экосистемы через компьютерное зрение

Проект: дообучение предобученного ResNet-18 (ImageNet) для классификации животных (птица, кошка, олень, собака, лягушка, лошадь) на снимках из CIFAR-10. Тематика соответствует мониторингу биоразнообразия: модель помогает автоматически распознавать животных на фото.

### Структура
- `main.ipynb` — весь пайплайн: загрузка данных, фильтрация классов, аугментации, обучение, метрики, визуализация.
- `requirements.txt` — зависимости.
- `venv/` — готовое окружение на Python 3.9.19.

### Быстрый старт
1) Активировать окружение:
   ```bash
   source "venv/bin/activate"
   ```
2) Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3) Открыть и запустить `main.ipynb` по порядку. Первый запуск скачает ~170 МБ CIFAR-10.

### Что делает ноутбук
- Оставляет в CIFAR-10 только классы животных, ограничивает размер выборки для быстрого прогона, делает стратифицированный split train/val/test.
- Аугментации + нормализация под ImageNet.
- Дообучает голову ResNet-18, сохраняет лучший чекпоинт по валидации.
- Считает accuracy, `classification_report`, нормированную confusion matrix и показывает примеры предсказаний.

### Ограничения и идеи улучшений
- CIFAR-10 — низкое разрешение (32×32), качество ограничено. Можно перейти на датасеты более высокого качества (например, iNaturalist, Caltech-UCSD Birds) для реальных сценариев мониторинга.
- Увеличить `max_train_per_class`, число эпох, настроить lr/scheduler, добавить mixup/cutmix для качества.
